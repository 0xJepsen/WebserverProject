## About The Project
Our webserver allows users to vote on cats by choosing the one they think is cutest.


<!-- USAGE -->
## USAGE
Ensure that flask is currently installed
```sh
pip install flask --user
pip install Flask-WTF
<!-- pip install WTForms -->
```
Run the program
```sh
python3 ./Webserver.py
```
